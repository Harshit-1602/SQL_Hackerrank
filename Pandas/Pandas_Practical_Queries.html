Pandas Practical Exercises (Top 50 Operations)

This document contains 50 essential Pandas operations that every Data Analyst should master. They are categorized to mirror common SQL workflows.

1. Basic Data Exploration

  1. Read CSV: df = pd.read_csv('worker.csv')
  2. First 5 Rows: df.head()
  3. Last 5 Rows: df.tail()
  4. Data Info: df.info()
  5. Statistical Summary: df.describe()
  6. Column Names: df.columns
  7. Shape of Data: df.shape
  8. Check for Nulls: df.isnull().sum()
  9. Unique Values: df['department'].unique()
  10. Value Counts: df['department'].value_counts()

2. Selection & Filtering

  1. Select Column: df['first_name']
  2. Select Multiple: df[['first_name', 'last_name']]
  3. Selection by Label (loc): df.loc[0:10, 'salary']
  4. Selection by Position (iloc): df.iloc[0:10, 0:3]
  5. Filter by Condition: df[df['salary'] > 100000]
  6. Filter (IN): df[df['first_name'].isin(['Vipul', 'Satish'])]
  7. Filter (NOT IN): df[~df['first_name'].isin(['Vipul', 'Satish'])]
  8. Between Range: df[df['salary'].between(100000, 500000)]
  9. Multiple Conditions: df[(df['salary'] > 50000) & (df['department'] == 'Admin')]
  10. Filter by Month/Year: df[(df['joining_date'].dt.year == 2014) & (df['joining_date'].dt.month == 2)]

3. String Manipulation

  1. Uppercase: df['first_name'].str.upper()
  2. Lowercase: df['first_name'].str.lower()
  3. Contains String: df[df['department'].str.contains('Admin')]
  4. Starts With: df[df['first_name'].str.startswith('A')]
  5. String Slicing: df['first_name'].str[:3]
  6. Replace String: df['first_name'].str.replace('a', 'A')
  7. Trim Whitespace: df['first_name'].str.strip()
  8. Concatenate Strings: df['first_name'] + ' ' + df['last_name']
  9. String Length: df['department'].str.len()
  10. Find Position: df['first_name'].str.find('B')

4. Data Cleaning & Transformation

  1. Drop Columns: df.drop(columns=['column_name'])
  2. Rename Columns: df.rename(columns={'old_name': 'new_name'})
  3. Change Data Type: df['salary'] = df['salary'].astype(float)
  4. To Datetime: df['joining_date'] = pd.to_datetime(df['joining_date'])
  5. Drop Duplicates: df.drop_duplicates()
  6. Fill Nulls: df.fillna(0)
  7. Drop Nulls: df.dropna()
  8. Reset Index: df.reset_index(drop=True)
  9. Sort Values: df.sort_values(by=['salary', 'first_name'], ascending=[False, True])
  10. Map Values: df['gender'].map({'M': 1, 'F': 0})

5. Aggregation & Grouping

  1. Group By Count: df.groupby('department').size()
  2. Group By Aggregation: df.groupby('department')['salary'].agg(['sum', 'mean', 'count'])
  3. Pivot Table: df.pivot_table(index='department', values='salary', aggfunc='mean')
  4. Cross Tabulation: pd.crosstab(df['department'], df['salary_bracket'])
  5. Apply Function: df['salary'].apply(lambda x: x * 1.1) # 10% raise

6. Merging & Advanced

  1. Merge (Join): pd.merge(df1, df2, on='worker_id', how='inner')
  2. Concatenate (Union): pd.concat([df1, df2], axis=0)
  3. Ranking: df['rank'] = df['salary'].rank(method='dense', ascending=False)
  4. Shift (Lag): df['prev_salary'] = df['salary'].shift(1)
  5. Rolling Mean: df['rolling_avg'] = df['salary'].rolling(window=7).mean()
