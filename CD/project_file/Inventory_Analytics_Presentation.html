🔍 Project Overview

The Vermin Inventory Analytics System is a powerful web-based application designed to streamline inventory data management and provide real-time analytics through interactive visualizations.

Problem Statement

  • Manual inventory tracking is error-prone and time-consuming
  • Difficulty in analyzing large datasets
  • Limited visibility into inventory trends
  • Need for automated data cleaning and processing

Solution

  • Automated data cleaning and preprocessing
  • User-friendly web interface for CRUD operations
  • SQL-based querying for custom analysis
  • Power BI integration for visual analytics

⚡ Key Features

📊 Data Processing

Automatic data cleaning, deduplication, and handling of missing values with smart defaults

➕➖ CRUD Operations

Complete Create, Read, Update, Delete functionality for all inventory records

🔍 SQL Query Interface

Run custom SQL queries directly on inventory data for advanced analytics

📈 Power BI Integration

Embed interactive Power BI dashboards and export data to Excel format

💾 Data Export

Download processed data as CSV files or bundle multiple tables as ZIP

🔗 Multi-Table Support

Handle multiple inventory sheets with unified interface

🔄 Data Flow & Processing

Data Pipeline

  • Input: Excel files with multiple inventory sheets
  • Cleaning: Automatic column standardization, duplicate removal, null handling
  • Storage: SQLite in-memory database for SQL queries
  • Output: CSV, ZIP, Power BI visualizations

Key Functions

  • clean_data() - Normalizes columns, removes duplicates, fills missing values
  • get_next_id_value() - Smart auto-increment for IDs
  • get_numeric_bounds_and_default() - Safe input validation
  • save_tables() - Multi-sheet Excel export
